Summary of ps command

ps command shows information about the processes being run on the system. It is a standard utility in POSIX, and most Linux distributions use ps implementation of procps package. A user can give command-line options to ps command to specify:

  • processes to display
  • display format

Default behavior

  • select processes that have the same EUID as the current user and associated with the same terminal as the invoker.
  • display the following information:
    • process ID
    • terminal associated with the process
    • cumulated CPU time
    • executable name
